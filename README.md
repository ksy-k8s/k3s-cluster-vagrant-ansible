# k3s-cluster-vagrant-ansible

Setup K3s cluster using Vagrant and Ansible.

## Configuration
- Node configuration : 1 server + 3 agent
- Provisioning : Vagrant + VirtualBox + Ansible
- OS : Ubuntu 22.04 LTS
